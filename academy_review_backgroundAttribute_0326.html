CSS - background 속성

background-color: red ; - 배경 색상 설정

background-image: url(경로) ; - 배경 이미지 설정

background-repeat: no-repeat ; - 배경 이미지 반복 설정

no-repeat 은 반복하지 않겠다라는 의미입니다.

background-size: cover ; - 배경 이미지 크기 설정

contain 은 내가 만든 박스의 세로 크기에 이미지가 맞춰져서 들어간다는 의미 입니다.

cover 는 내가 만든 박스의 가로 크기에 이미지가 맞춰져서 의미 입니다.

background-position: center ; - 배경 이미지 위치 설정

center / right / left / px / % 를 통해 설정 가능합니다.

background-attachment: fixed ; - 스크롤바의 움직임에 따라 배경이미지가 변경됨

scroll : 선택한 요소와 같이 움직입니다. 내용을 스크롤하면 배경 이미지는 스크롤되지 않습니다. (기본값)

fixed : 움직이지 않습니다.
