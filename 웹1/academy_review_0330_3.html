포지션 - position

static - 모든 태그는 기본적으로 position : static 속성이다.

relative - 현재 위치를 기준으로 이동하는 속성 으로 absolute 속성의 기준점이 되는 속성

absolute - relative 를 기준점으로 이동하며 다른 엘리먼트들 위에 위치하게 되는 유령화시키는 속성

absolute 속성을 사용할 경우에는 부모 엘리먼트에 relative 를 꼭 넣어야 한다.
만약 부모 엘리먼트에 relative 가 없다면 그 상위 부모로 찾게된다.
아무것도 relative 가 없다면 body 를 기준으로 움직이게 된다.

fixed - 윈도우를 기준으로 움직이며 스크롤이 변경 되어도 위치가 변하지 않는 속성
