마우스 오버 속성 - 마우스를 올렸을때 css 변경되는것.

사용방법 : css 에서 클래스 명뒤에 :hover 라고 입력

ex) .box:hover {width: 300px;}

예시

200*200 박스 입니다.

transition 속성 - css 가 변경될때 자연스러운 애니메이션 구현

사용방법 : transition : 1s ; 변경되는 시간 설정

ex) .box {transition : 1s ;}

예시

200*200 박스 입니다.

font 관련 속성

font-size: 24px ;

font-weight: bolder ;

폰트웨이트는 기본 = normal, 100~900, bold, bolder

font-family: 'Nanum Pen Script', cursive;

구글 폰트 에서 선택후 링크 걸고 변경

text-shadow: 10px 10px 10px black ;

float 속성

참고 사이트

float : left ; (요소들을 왼쪽으로 정리한다)

float : right ; (요소들을 오른쪽으로 정리한다)

float 속성을 준 부모 태그에는 무조건 아래의 속성을 주어 레이아웃이 틀어지지 않도록 한다.

기본적으로 row 라는 클래스를 만들어 넣어줌

.row::after{

clear:both;

display:block;

content:"";

}

display 속성

참고 사이트

display : block ;

display 속성이 block으로 지정된 엘리먼트는 전후 줄바꿈이 들어가 다른 엘리먼트들을 다른 줄로 밀어내고 혼자 한 줄을 차지합니다.

기본 요소가 block 인 엘리먼트들 : h1~6, p , div , ul ,li

display : inline ;

display 속성이 inline으로 지정된 엘리먼트는 전후 줄바꿈 없이 한 줄에 다른 엘리먼트들과 나란히 배치됩니다.

※ inline 엘리먼트를 사용할 때 주의할 점은, width와 height 속성을 지정해도 무시된다는 것입니다. 왜냐하면 해당 태그가 마크업하고 있는 컨텐트의 크기 만큼만 공간을 차지하도록 되어 있기 때문입니다. 또한, margin과 padding 속성은 좌우 간격만 반영이 되고, 상하 간격은 반영이 되지 않습니다.

기본 요소가 inline 인 엘리먼트들 : a , span , img

저는 block 입니다.

저는 inline 입니다.

display : none ;

안보이게 하는 태그입니다.
